About AiiDAlab

AiiDAlab is an all-in-one Jupyter-based web platform for running materials simulation workflows powered by AiiDA. AiiDAlab can be used both for research and education.

About AiiDA

AiiDA is an open-source workflow engine with built-in HPC support, automatic full data provenance, and a rich plugin system for extending and scaling capabilities.
